# 🎮 NextGenDominators ROM Hub

A sleek, modern web-based ROM hub for classic Pokémon games.\
Built using pure **HTML, CSS, and JavaScript**, this project provides a
visually appealing interface for browsing and accessing game versions.

------------------------------------------------------------------------

## ✨ Features

-   🎨 Modern dark UI with smooth animations\
-   📱 Fully responsive grid layout\
-   🧩 Dynamic modal download system\
-   ⚡ Interactive user experience

------------------------------------------------------------------------

## 🗂️ Project Structure

NextGenDominators-ROM-Hub/ │── index.html │── photo/ │── README.md

------------------------------------------------------------------------

## 🚀 How to Use

1.  Download or clone the repository\
2.  Open the folder\
3.  Run `index.html` in your browser

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   HTML5\
-   CSS3\
-   JavaScript

------------------------------------------------------------------------

## 📦 Game Support

-   Gen 1: Red, Blue, Green, Yellow\
-   Gen 2: Gold, Silver, Crystal\
-   Gen 3: Ruby, Sapphire, Emerald, FireRed, LeafGreen
